Frequently Asked Questions

API Keys

How do I get a Gemini API key?

1. Go to Google AI Studio
2. Click "Get API Key"
3. Free tier allows 60 requests per minute

Is OpenAI free to use?

No, OpenAI requires a paid account. Use Gemini for free access.

Performance

Why is my AI slow to respond?

Try these optimizations:
• Use gemini-1.5-flash model
• Reduce response length in system prompts
• Check your internet connection

How do I reset the conversation memory?

Use the reset memory block or restart your project.

Troubleshooting

I'm getting "API Error" messages

1. Verify your API key is correctly set
2. Check the model compatibility
3. Ensure you're not exceeding rate limits

Blocks aren't showing up in TurboWarp

1. Reload the extension
2. Clear browser cache
3. Re-download ai.js from our docs
